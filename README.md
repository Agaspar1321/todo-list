Overview - This is a To-Do-List application with a drop and drag feature. Users enter tasks and can use buttons to clear all items(Clear Button) or delete individual tasks with button right next to task(Delete Button). You are also able to move each task up and down by dragging it.

Features:
- Add tasks: Enter new items
- Delete individual tasks: Remove items one at a time
- Clear all tasks: Reset list
- Drag and drop: Reorder tasks by moving them up or down
- Persistent storage: Saves tasks in localStorage

Tech Stach: HTML - Structue | CSS - Design | JavaScript - Logic + localStorage

Live Demo - https://agaspar1321.github.io/todo-list/

ScreenShot:![App Screenshot](image-1.png)

How it works:
1. User enter a task into input field.
2. The tasks are saved to local storage.
3. Tasks are reloaded after page refreshes
4. User can delete individual items or clear them all.
5. Drag-and-drop allows reordering
